# App Gatos Nocturnos

Aplicación Next.js para explorar la demanda nocturna no Metro en Santiago con celdas H3, corredores OD, estaciones y métricas territoriales.

## Desarrollo

```bash
npm install
npm run dev
```

Luego abre `http://127.0.0.1:3000`.

## Producción

```bash
npm run build
npm start
```
